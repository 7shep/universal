# Universal demo site

This is a minimal, static React/Vite project intended for an end-to-end Universal MCP demonstration.

```bash
pnpm install
pnpm --filter @universal/demo-site dev
```

Vite serves the demo at `http://localhost:5175`. Use the Universal MCP before substantially redesigning the page, then review `src/app.tsx` and `src/styles.css` with `review_implementation`.
