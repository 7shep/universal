# Product

## Register

product

## Users

Developers working locally with coding agents who need to turn a rough product idea into a deliberately art-directed React interface. Their immediate job is to establish a credible design direction before implementation begins.

## Product Purpose

Design Studio is an AI Art Director for React applications. It will guide a design-first workflow that produces design specifications, compositions, critique, and previewable output for existing coding agents. Success is an interface that feels specifically composed rather than statistically generated.

## Brand Personality

Exacting, editorial, and constructive. The product should make creative direction feel legible and practical without acting like a visual playground detached from code.

## Anti-references

Generic AI website builders; purple-gradient SaaS dashboards; interchangeable card grids; configuration-heavy tools that make composition feel secondary to controls.

## Design Principles

- Start with a design decision, not a component inventory.
- Make composition a first-class, inspectable artifact.
- Preserve creative intent through every handoff to code.
- Give critique in concrete, actionable language.
- Keep the local workflow fast and transparent.

## Accessibility & Inclusion

Target WCAG 2.2 AA, keyboard-accessible controls, visible focus states, and reduced-motion support. Design guidance should never depend on color alone.
